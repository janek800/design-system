# Color System

**The color system is crafted according to WCAG 2.0 Accessibility guidelines, reference below ðŸ‘‡**

#### AA

WCAG 2.0 level AA requires a contrast ratio of at least 4.5:1 for normal text and 3:1 for large text.<br/> WCAG 2.1 requires a contrast ratio of at least 3:1 for graphics and user interface components (such as form input borders).

#### AAA

WCAG Level AAA requires a contrast ratio of at least 7:1 for normal text and 4.5:1 for large text.<br/> Large text is defined as 14 point (typically 18.66px) and bold or larger, or 18 point (typically 24px) or larger.

ðŸ“’ Note

1. Use the primary color for interactive text / Hyperlink.
2. Check color contrast at, <https://contrastchecker.com/>
